All Endpoints Available for this API

  • GET /users
  • GET /users/:id
  • POST /users
  • PATCH /users
  • PUT /users/:id
  • DELETE /users/:id
  • GET /users/:id/cart
  • POST /users/:id/cart
  • PUT /users/:id/cart
  • DELETE /users/:id/cart

Products:

  • GET /products
  • GET /products/:id
  • POST /products
  • PUT /products/:id
  • DELETE /products/:id
